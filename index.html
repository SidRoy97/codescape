CodeReach

Real-time code quality, workspace-wide taint analysis, AI-powered fixes, and an interactive code impact graph for VS Code. No server required.

🕸️

Code Impact Graph

Callers, callees, and blast radius for any function — across your whole project. Import-aware: uses actual import statements to eliminate false edges.

🔬

Workspace Taint Analysis

Follows untrusted input from HTTP params, forms, and CLI args to SQL, shell, and HTML sinks — across file boundaries, for Python, Java, and JS/TS.

🔒

Security Analysis

SQL injection, XSS, hardcoded secrets, command injection, weak crypto, path traversal — static rules that run instantly with no AI needed.

🧹

Code Quality & SOLID

God classes, deep nesting, callback hell, boolean flags, chained ternaries, too many parameters, TypeScript any usage.

⚛️

React Anti-Patterns

Rules of Hooks, async useEffect, state mutations, re-render traps, deprecated lifecycle methods — 30+ checks.

🧠

Understanding Doc

Every symbol, a one-line AI summary, and its caller/callee relationships — structured context to onboard a human or give an LLM accurate codebase knowledge.

✍️

Auto-Comment

Inserts JSDoc, Google-style docstrings, and Javadoc above uncommented functions across the workspace — with or without AI.

🤖

AI Analysis (Optional)

Catches what patterns can't — logic errors, wrong algorithms, missing edge cases. Static rules run instantly on save; AI runs as a second pass when enabled. One-click Fix with AI (diff preview) and Explain this Issue. Off by default, nothing sent to a model unless you turn it on.

📊

Dashboard & Reports

Sidebar dashboard scoped to file or workspace. Export codereach-issues.md and .json grouped by file and function for sharing or CI integration.

🔬 Workspace Taint Analysis

Traces untrusted data from entry points across file boundaries to dangerous sinks — for Python, Java, and JavaScript/TypeScript projects.

Sources

HTTP params, form fields, URL path params, CLI args, env vars — aiohttp, Flask, Django, FastAPI, Express, Next.js, Spring, JAX-RS, Servlet.

Sinks

SQL injection, XSS, command injection, SSTI, open redirect, deserialization, prototype pollution, path traversal, LDAP injection, XXE.

Smart Filtering

Sanitizer-aware, cross-file BFS propagation up to 4 hops. Skips static/, vendor/, *.min.js, target/, venv/, migrations/ automatically.
